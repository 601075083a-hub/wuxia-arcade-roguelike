# Wuxia Arcade Roguelike

武侠街机 Roguelike 是一个以武侠为题材、以 Deckbuilder 为构建骨架、以街机搓招为核心出招方式的轻量 Roguelike 游戏项目。

当前阶段目标是先完成一个“完整但窄”的可玩 Demo，用最小内容验证核心体验：

- 玩家通过方向和攻击碎片组合招式。
- 外功提供新搓招配方和类型增幅。
- 内功改写战斗规则，形成构建变化。
- 跑图、战斗、奖励、商店、事件和 Boss 构成一局完整闭环。

## Demo 范围

首版 Demo 采用“完整但窄”的范围：

- 1 个主角
- 6 种基础碎片
- 10-12 个基础或外功招式
- 6-8 个外功
- 6-8 个内功
- 5 个普通或精英敌人
- 1 个 Boss
- 1 套基础跑图流程

商业化、多角色、完整天赋树和大量内容扩展暂不进入首版范围。

## 项目状态

当前处于产品规格整理阶段，尚未确定最终技术栈。

后续会根据 Demo 目标选择实现方式，例如微信小游戏原生、Cocos、Laya 或 Web 原型。
