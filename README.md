# Wuxia Arcade Roguelike

武侠街机 Roguelike 是一个以武侠为题材、以 Deckbuilder 为构建骨架、以街机搓招为核心出招方式的轻量 Roguelike 游戏项目。

当前阶段目标是先完成一个“完整但窄”的可玩 Demo，用最小内容验证核心体验：

- 玩家通过方向和攻击碎片组合招式。
- 外功提供新搓招配方和类型增幅。
- 内功改写战斗规则，形成构建变化。
- 跑图、战斗、奖励、商店、事件和 Boss 构成一局完整闭环。

## Demo 范围

首版 Demo 采用“完整但窄”的范围：

- 1 个主角
- 6 种基础碎片
- 10-12 个基础或外功招式
- 6-8 个外功
- 6-8 个内功
- 5 个普通或精英敌人
- 1 个 Boss
- 1 套基础跑图流程

商业化、多角色、完整天赋树和大量内容扩展暂不进入首版范围。

## 项目状态

当前已进入原生微信小游戏 Demo 阶段。

## 运行方式

1. 打开微信开发者工具。
2. 选择“小游戏”项目。
3. 导入本仓库根目录。
4. 使用测试 AppID 或游客模式运行。

首版使用 Canvas 2D 和原生 JavaScript，不依赖构建工具。

## 工程结构

- `game.js`：小游戏入口。
- `src/`：核心逻辑、状态机和 Canvas 渲染。
- `data/`：碎片、招式、外功、内功、敌人和跑图配置。
- `assets/`：后续美术资源预留，目前使用 Canvas 占位绘制。
